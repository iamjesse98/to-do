Welcome to to-do app

To Read Manual
------------------------
Use `python3 todo.py`

Adding user
------------------------
Use `python3 todo.py create <name>`
 
Adding an item
------------------------
Use `python3 todo.py add <item> <name>`

Removing an item
------------------------
Use `python3 todo.py remove <item> <name>`

Displaying user's items
------------------------ 
Use `python3 todo.py list <name>`

Delete a User
------------------------
Use `python3 todo.py delete <name>`
